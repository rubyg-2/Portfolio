Projects

Project 1: Autonomous Maze-Navigating Robot

This was a semester-long team project focused on designing, building, coding, testing, and demonstrating an autonomous robot. Working with a team of three other members, we developed a robot capable of navigating a maze to transport a disabled person while adhering to traffic control rules. The robot used a color sensor to detect path indicators: Red: Stop for a second before continuing. Blue: Switch to manual mode (after beeping twice) for a teammate to assist with passenger transfer. Yellow: Designated drop-off zone, switching to manual mode for final assistance. The entire system was programmed and controlled using MATLAB, ensuring seamless autonomous and manual transitions.

Project 2: Interactive Q&A Application

This is an ongoing team project designed to create a user-friendly Q&A platform for students, built using JavaFX for the GUI and a database to store all user data persistently. The application enables students to ask and answer questions while providing features for private messaging, role management, and search functionality. Key Features: User Authentication: Users can create an account and log in, ensuring their data is saved even after closing the application. Role Management: Admins can assign and manage user roles while viewing all registered users. Q&A Interface: Students can post, answer, and mark questions as resolved or unresolved. Private Messaging: Users can send private responses to others for direct discussions. Search Functionality: A built-in search feature allows users to quickly find relevant questions and answers. This application promotes a collaborative learning environment, ensuring students can engage in meaningful discussions while keeping track of their interactions and progress.

Project 3: Women in Tech

This website is dedicated to promoting and encouraging more women to pursue careers in technology. It highlights the gender gap in the industry, discusses its importance, and provides valuable resources for young girls to get involved in tech. Through shared information and accessible opportunities, the goal is to inspire and support the next generation of women in technology.

Project 4: Book Management System

The program features a menu-driven interface, enabling users to interact with the system through commands. Users can store, retrieve, update, and delete book records based on input choices. This program implements a linked-list-based Book Management System in C++. Each book is uniquely identified by its ID, a combination of its title and edition, and stored in alphabetical order.
